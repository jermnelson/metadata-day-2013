BIBFRAME & the ADR

The Redis Library Services Platform already has a number of HTML5 apps for both public and staff work-flows to the ADR. Adding Fedora Objects into the BIBFRAME Redis Datastore is being tested with public access through the Aristotle Library Apps's Discovery App.

Embedding BIBFRAME

CC has experimented with embedding LinkedData (schema.org) into separate namespaces in the MODS metadata for ADR subjects. While the current ADR Fedora Commons supports this inclusion of other metatadata vocabularies into MODS, Islandora would need to be modified to provide a public interface to such vocabularies.

Redis Library Services Platform

The Redis Library Services Platform natively supports HTML5 apps that interact directly with Fedora. These include a batch creation tool, a PID mover app, and an app for students to directly submit their thesis and supporting datasets, video, or audio to the ADR. The platform is also actively improving the MARC-to-BIBFRAME and MODS-to-BIBFRAME ingestion workflows.

While still in the planning stages, we are looking at an Alliance-wide Redis Library Services Platform that would provide each institution a suite of supporting apps for their use including both the BIBFRAME Discovery App and the Fedora Utilities App.
